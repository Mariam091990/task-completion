Question-2: what are the difference between innerHTML, innerText and textContent?

The innerHTML property returns the full content including all the HTML tags.
The innerText property returns the content as appears on the screen. It ignores all the HTML tags and the hidden element (with display set to none).
The textContent property returns all including hidden contents as it in the HTML markup and ignores the HTML tags.

Question-3: what is event delegation in the DOM?

The idea of "delegate" .. the handling of an event to a different element instead of the actual element that received the event.

Question-4: what is event bubbling in the DOM?

when an element receives an event, and that event transmitted to its parent elements in the DOM tree until it gets to the root element.
